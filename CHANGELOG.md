# Changelog

All notable changes to the "vscode-md-browser" extension will be documented in this file.

## [0.1.1] - 2026-03-17
- Fixed opening multiple links in one action.
- `md-browser.openUrl` now supports multiple cursors/selections and URL lists.
- Improved URL input parsing (comma/newline/space-separated URLs).

## [0.1.0] - Initial Release
- Open URLs from Markdown files in VS Code's embedded Simple Browser.
- Dedicated embedded YouTube video player with autoplay.
- Support for various link formats (`[text](url)`, `[url](url)`, bare URLs).
- Context menu options and `Alt+Enter` keyboard shortcut.
